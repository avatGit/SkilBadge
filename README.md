# SkillBadge - Certification NFT des compétences autodidactes

##  Démo Live
- Frontend : https://skillbadge.vercel.app
- Contrat testnet : https://amoy.polygonscan.com/address/0x...
- Vidéo de démo : [Lien Loom]

##  Stack Technique
- Frontend : React + Vite + TypeScript + Tailwind + Framer Motion
- Web3 : wagmi + viem + MetaMask
- Blockchain : Polygon Amoy (ERC-1155)
- Stockage : IPFS (Pinata)

##  Installation & Déploiement
1. `git clone ...`
2. `cd client && npm install`
3. Copier `.env.example` vers `.env`
4. `npm run dev`
